# LLM Text Summarization using BART

This project implements an abstractive text summarization system using a pre-trained BART model to summarize user reviews.

## Project Structure
- notebooks/: Jupyter notebook implementation
- data/: Dataset files
- README.md: Documentation

## Model
- facebook/bart-large-cnn
- HuggingFace Transformers

## Evaluation
Summaries were evaluated qualitatively for coherence and coverage.

## Future Work
Fine-tuning, multilingual summarization, sentiment-aware summaries.
